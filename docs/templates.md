# Templates

Builder includes workflow templates for supported project types.

## Available Templates

| Template | Location | Project Type |
|----------|----------|-------------|
| Native iOS | `templates/ios-xcode.yml` | Xcode projects |
| Flutter | `templates/ios-flutter.yml` | Flutter projects |
| React Native | `templates/ios-react-native.yml` | React Native projects |
| Expo | Embedded in binary | Expo projects |
| Capacitor | Embedded in binary | Capacitor projects |
| Unity | Embedded in binary | Unity projects |
| Unreal | Embedded in binary | Unreal projects |

## Adding Custom Templates

Place custom workflow files in `.github/workflows/` in your repository.
