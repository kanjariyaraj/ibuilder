==================================================
PRE-IMPLEMENTATION
==================================================

Before coding:

1. Pull latest repository.
2. Read all reports.
3. Read architecture docs.
4. Read build engine.
5. Read signing engine.
6. Read Flutter workflow.
7. Read React Native workflow.
8. Verify build passes.
9. Verify tests pass.
10. Create implementation plan.

Never start coding immediately.

Workflow:

ANALYZE
→ PLAN
→ TODO
→ IMPLEMENT
→ TEST
→ FIX
→ RETEST
→ DOCUMENT
→ REPORT
→ COMMIT
→ PUSH
