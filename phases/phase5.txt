PHASE 5
REMOTE IOS BUILD ENGINE
==================================================

CRITICAL PHASE

This phase implements the core value of iBuilder.

Goal:

Allow users to build iOS applications remotely
using GitHub-hosted macOS runners and download
the resulting IPA locally.
