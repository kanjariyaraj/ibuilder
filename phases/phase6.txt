IOS SIGNING MANAGEMENT SYSTEM
==================================================

CRITICAL PHASE

This phase implements certificate and provisioning
profile management.

Allow developers to configure, validate and manage
iOS signing without manually editing GitHub workflows.

==================================================

Implement complete signing management.

Users should be able to:

Upload provisioning profiles

Validate signing

Verify expiration

Manage secrets

Diagnose signing issues

==================================================
COMMANDS
==================================================

ibuilder signing setup

ibuilder signing validate

ibuilder signing doctor

ibuilder signing status

ibuilder signing remove

ibuilder signing sync

ibuilder signing export

==================================================
SIGNING FLOW
==================================================

1. User runs setup.
2. Select certificate.
3. Select provisioning profile.
4. Enter password.
5. Validate files.
6. Encode securely.
7. Upload GitHub secrets.
8. Update config.
9. Verify installation.
10. Generate report.
